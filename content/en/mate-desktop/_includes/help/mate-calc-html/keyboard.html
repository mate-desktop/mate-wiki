Using the Keyboard

All mathematical equations can be entered using the keyboard.

The following key combinations can be used to enter keys that may not be available on your keyboard.

×

*

÷

/

^

* twice

√

Ctrl+R

π

Ctrl+P

To enter superscript numbers use Ctrl+number, for subscript use Alt+number.

When in programming mode, to cycle between base digits use:

Binary

Ctrl+B

Octal

Ctrl+O

Decimal

Ctrl+D

Hexadecimal

Ctrl+H

To cycle between history entries you may use:

Previous Entry

Alt+Up

Next Entry

Alt+Down
